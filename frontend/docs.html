ToknNews System Architecture

Welcome to the ToknNews technical overview. This system is designed as a fully autonomous AI-powered broadcast pipeline capable of ingesting, enriching, prioritizing, scripting, and broadcasting crypto and financial market news.

Core Pipelines

  • Ingestion v2: Multi-source RSS & API collector
  • Enrichment v2: GPT-driven summaries and domains
  • Meta-Enrichment: Contextual narrative & risk mapping
  • StoryBank: Long-term AI memory storage
  • PDv4: Persona Director with intelligent casting
  • Timeline Builder v4: Multi-anchor, mode-aware scripts

Show Modes

  • Breaking
  • Morning Brief
  • News
  • Heavy News
  • Deep Dive
  • Chaos Mode
